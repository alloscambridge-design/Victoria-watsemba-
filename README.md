# Allos Cambridge Online School Website

**A professional, responsive website for Allos Cambridge Online Learning & Homeschooling**

## 🌐 Features

✅ **Responsive Design** - Mobile-first approach, works on all devices  
✅ **SEO Optimized** - Meta tags for search engines  
✅ **Interactive Elements** - Smooth scrolling, accordion FAQs, mobile menu  
✅ **Contact Forms** - Application and inquiry submission  
✅ **Professional Branding** - Clean design with Cambridge school colors  
✅ **Fast Loading** - Lightweight HTML/CSS with minimal dependencies  
✅ **WhatsApp Integration** - Quick contact button for messaging  

## 📋 Sections

1. **Navigation** - Sticky header with mobile menu
2. **Hero Section** - Compelling headline and CTAs
3. **About** - School mission and overview
4. **Why Choose Us** - Key differentiators
5. **Programs** - Course offerings (Primary, IGCSE, A-Levels, Tutoring)
6. **How It Works** - Step-by-step process
7. **Pricing** - Transparent pricing for each program
8. **Testimonials** - Student success stories
9. **FAQ** - Common questions with interactive accordion
10. **Application Form** - Student admission form
11. **Contact** - Contact information and inquiry form
12. **Footer** - Copyright and links

## 🚀 Deployment Options

### GitHub Pages (Free)
```bash
# Push to GitHub and enable GitHub Pages in repository settings
# Your site will be live at: https://alloscambridge-design.github.io/Victoria-watsemba-/
```

### Netlify (Recommended)
1. Connect GitHub repository
2. Deploy automatically on every push
3. Free SSL certificate
4. Enhanced form handling

### Vercel
1. Import GitHub repository
2. Deploy with one click
3. Serverless functions for backend

## 🔧 Customization

### Update Contact Information
Find and replace:
- `alloscambridge@gmail.com` → Your email
- `+256 706 901 823` → Your WhatsApp number
- `Online Worldwide 🌍` → Your location

### Change Colors
Edit CSS variables in `<style>` section:
```css
:root {
  --blue: #1e3a8a;      /* Primary color */
  --red: #dc2626;       /* Accent color */
}
```

### Add Your Logo
Replace "ALLOS CAMBRIDGE" text with an `<img>` tag.

## 📧 Form Integration
